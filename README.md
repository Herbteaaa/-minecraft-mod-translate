# Minecraft Mod 日本語化自動化ツール (開発中 v7)

※現在も改善を継続している開発中プロジェクトです

## 概要
MinecraftのModに含まれる英語テキストを自動で日本語化するツールです。  
複数のModを一括処理し、プレイ時の理解を向上させることを目的としています。

## 背景
Modを導入して遊ぶ中で、日本語に対応していないものが多く、  
内容の理解が難しいという課題を感じました。  
既存の日本語化手段を調査しましたが、汎用的に対応できるツールがなかったため、  
自作することにしました。

## 主な機能
- Modファイル（.jar）の展開・再構築
- JSON / lang ファイルの自動翻訳
- 複数Modの一括処理
- 翻訳結果の上書き保存

## 使用技術
- Python
- Argos Translate（初期版）
- NLLB-200（改良版）
- JSON処理 / ファイルI/O / zip操作

## ファイル構成
- auto_translate_v7.py：改良版（NLLB使用）
- auto_translate_v7_argos.py：初期版（Argos使用）

## 使用方法
1. 対象のModファイル（.jar）を指定のmodsフォルダに配置
2. スクリプトを実行
3. 翻訳済みModが出力フォルダに生成される

## 実行環境
- Python 3.x
- 必要ライブラリ（transformers, torch など）

※詳細はコード内参照

## 工夫点
- 翻訳精度向上のためArgos → NLLBへ変更
- 固有名詞やカタカナを保持するためのグロッサリ導入
- プレースホルダ保護によりゲーム内表示を破壊しない設計
- 文章分割により文脈を考慮した翻訳
- 大量ファイル処理に対応するための効率化

プレイヤーが違和感なくゲーム体験を得られることを重視して改善を行っています。

## 技術的なポイント
- jar形式の展開・再構築処理を実装
- JSON / lang形式の構造を維持したまま翻訳処理を適用
- プレースホルダ文字列を保護しながらのテキスト変換

## 成果
- 初期版（Argosモデル）において、複数（数十以上）のModに適用し、日本語化動作および内容理解の向上を確認
- 英語のみの状態と比較して、プレイ中の操作・仕様の把握が容易になった
- 一括処理により複数Modの日本語化が可能となり、作業効率を改善

※改良版（v7）は開発中のため継続検証中ですが、翻訳精度の改善が期待されます。

## 課題・改善点
- 大量ファイル処理に時間がかかるため高速化が必要（特に複数Mod同時処理時）
- 翻訳精度のさらなる向上
- 一部文脈での誤訳の抑制
